# People Manager Playground

A personal sandbox repo based on the persons-api project.
Used for experimenting, testing new ideas, and trying out new patterns before applying them to production projects.

---

## What this is

This is NOT a production repo.
This is a place to:
- try new libraries
- experiment with new patterns
- test ideas before applying them to real projects
- break things on purpose and learn from it

---

## Stack

- TypeScript
- Node.js + Express
- Jest + Supertest
- ESLint + Prettier

---

## Installation

```bash
git clone https://github.com/Adam-hash-a11y/People-Manager-playground.git
cd people-manager-playground
npm install
```

## Run

```bash
npm run dev
```

## Tests

```bash
npm run test
```
